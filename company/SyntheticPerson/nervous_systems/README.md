# Cognitive Nervous Systems

The 12 nervous systems form the cognitive architecture of SyntheticPerson.ai.
Each is a **self-sufficient cognitive subsystem** with perception, reasoning,
memory access, coordination, and learning. None of them is a department; each
is a continuous process running over the same shared substrate.

## The Twelve

| System | Function in one line |
|---|---|
| `organizational_cognition_system` | Meta-cognition: the org thinking about itself |
| `market_intelligence_system` | Sensing markets, customers, competition |
| `synthetic_research_system` | Generating and validating synthetic research |
| `simulation_system` | Running simulations across domains |
| `product_evolution_system` | Sensing product reality and evolving offering |
| `platform_intelligence_system` | Sensing the SyntheticPerson platform itself |
| `organizational_memory_system` | Curating and serving the memory substrate |
| `strategic_intelligence_system` | Long-horizon strategic reasoning |
| `human_interaction_system` | Mediating human-system interaction |
| `adaptive_coordination_system` | Distributed coordination protocols |
| `learning_system` | Continuous evolution and adaptation |
| `governance_system` | Active governance and policy enforcement |


## Inter-System Topology

There is no master. The 12 are peers connected by:

- The shared memory substrate (read/write)
- The cognitive event bus (publish/subscribe)
- Governance policies (constraint propagation)
- Simulation invocation contracts

The `governance_system` and `learning_system` operate as **first-among-peers**
on policy and evolution; they do not command, they govern.
