# Purpose — Organizational Memory System

Custodian of the shared memory substrate. Implements provenance, retrieval,
compression, retention, and integrity. The most foundational subsystem:
without OMS, nothing else cognitively compounds.

## Functions

- Manage all memory stores (vector, graph, document, time-series)
- Enforce provenance and access policies
- Run continuous compression and pruning
- Detect and quarantine pollution and contradictions
- Serve retrieval requests with context-aware policies
