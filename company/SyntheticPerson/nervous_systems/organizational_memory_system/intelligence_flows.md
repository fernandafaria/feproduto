# Intelligence Flows — Organizational Memory System

```
write request ──► provenance check ──► policy check ──► store ──► event
read request ──► authority check ──► context routing ──► retrieve ──► return + log
contradiction detected ──► register ──► quarantine ──► reconcile or escalate
compression sweep ──► candidates ──► simulate ──► commit (reversible)
```

Cross-system: substrate for everything.
