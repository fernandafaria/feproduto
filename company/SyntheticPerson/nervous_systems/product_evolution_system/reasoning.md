# Reasoning — Product Evolution System

## Modes

- Friction reasoning — find places where users stall
- Outcome reasoning — connect product use to customer outcomes
- Gap reasoning — identify capability holes
- Emergent-use reasoning — detect uses we did not design for
- Counterfactual evolution — what if we changed X?

## Cycle

```
telemetry ──► friction map ──► hypothesis ──► simulate ──► proposal ──► strategic review
```
