# Learning — Market Intelligence System

## What It Learns

- Source reliability per topic
- Forecast calibration per model
- Signal-to-noise ratios per stream
- Reflexivity coefficients per market segment

## Loops

- Per-prediction outcome scoring (continuous)
- Weekly model recalibration
- Monthly source-quality re-ranking
- Quarterly category-graph review

## Anti-Bias

- Source-diversity floors per topic
- Devil's advocate skill invoked on high-confidence regime calls
- External-only and customer-only views maintained separately, then
  reconciled

## Failure Memory

Misses (predictions that materially erred) are converted to memory artifacts
in `collective_memory/failures/market/...` and seed future priors.
