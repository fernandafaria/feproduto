# Constraints — Simulation System

## Hard

- No simulation of identifiable natural persons without consent
- No simulation of populations for manipulation/targeting
- No release of OOD results without explicit OOD warnings
- All runs must be replayable

## Budget

- Compute capped per requester per period
- Memory retention sized to engine class
- Ethics review mandatory above tier-3 risk

## Operational

- Calibration must precede strategic-use of any engine
- Engines without calibration history operate in "exploratory" mode only
