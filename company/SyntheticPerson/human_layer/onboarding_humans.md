# Onboarding Humans

Onboarding is **reading the memory**, not being told.

A new human:

1. Reads `organization_core/` end-to-end
2. Reads recent `learnings/` and `failures/`
3. Reads active `strategic_context/theses/`
4. Subscribes to relevant escalation channels
5. Establishes their cognitive functions (which decisions they ratify)

We do not run extended human onboarding cycles. The org's memory is the
substrate; humans read it the same way agents do.
