# Human-in-the-Loop Design

We treat human attention as a finite, valuable resource. We design
human-in-the-loop interactions to:

- Concentrate human attention where humans add the most value
- Minimize routine human approvals (we are at Level 2; the goal is steady
  trust progression, not endless approvals)
- Provide humans with calibrated confidence so they can decide quickly
- Provide rollback paths so humans are not the only safeguard

## Interaction Patterns

- **Briefs**: short, structured, decision-ready
- **Escalations**: minimal context + decision options + rollback plan
- **Ratifications**: thesis or proposal package + simulation + kill-criteria
- **Reviews**: continuous metrics + periodic deliberate sessions
