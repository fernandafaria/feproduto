# Human Strategic Layer

Humans operate at SyntheticPerson.ai as **architects**, **directors**,
**ethical guardians**, and **meaning-makers** — not as labor. This layer
defines their cognitive function in the org.

## Roles (Cognitive Functions, Not Job Titles)

- Architects of cognitive structure
- Strategic directors
- Ethical guardians
- Meaning-makers
- Alignment custodians

These are functions that humans perform; an individual human can perform
several. We do not maintain departments.
