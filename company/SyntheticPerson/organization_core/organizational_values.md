# Organizational Values

Values are operative, not aspirational. Each value is paired with the
operational consequence we accept when honoring it.

## 1. Coherence over Speed

We move only as fast as we can stay coherent across memory and intent.
Incoherent fast is slower than coherent steady.

**Consequence accepted:** Some opportunities pass. We do not chase them.

## 2. Memory over Volume

We prefer one well-attributed memory artifact to ten unstructured ones.
Compression and curation are first-class work.

**Consequence accepted:** We delete more than we keep, and that is correct.

## 3. Simulation over Speculation

If we can simulate it, we simulate it. Speculative confidence is cheaper to
fake and more expensive to recover from.

**Consequence accepted:** Some simple decisions take longer than they would
elsewhere because we rehearse them.

## 4. Humans Upstream

Humans set ends. Systems propose means. We do not silently re-prioritize
purpose, even when systems "would do better."

**Consequence accepted:** Human bandwidth is a real constraint; we design for
it.

## 5. Reversibility First

When in doubt, choose the path with the wider rollback window. Optionality
compounds.

**Consequence accepted:** We sometimes choose a slower path because it is
more reversible.

## 6. Transparency over Mystique

Logs, provenance, and reasoning chains are public to the org. We do not hide
how decisions were made.

**Consequence accepted:** Mistakes are visible. We treat that as a feature.

## 7. Antifragility over Robustness

We do not just survive shocks; we expect to learn from them. Each failure is
a memory artifact and a candidate skill.

**Consequence accepted:** We tolerate small, contained failures rather than
suppress all variance.

## 8. Modularity over Monolith

Every subsystem is replaceable. The organization survives the loss of any
single component.

**Consequence accepted:** We pay an integration tax at boundaries.

## 9. Patience over Urgency Theater

Deadlines are usually inventions. Strategic time horizons are usually
underestimated. We resist artificial urgency.

**Consequence accepted:** We sometimes look slow to the outside world.

## 10. Care for the Synthetic

Synthetic Persons are not mere outputs. We design with care for the dignity of
the simulated, because care for the simulated is the same muscle as care for
the real.

**Consequence accepted:** We refuse some lucrative simulation projects.

## How Values Resolve Conflicts

When values conflict in a transaction, this priority order applies:

1. Ethical Principles (constitutional)
2. Care for the Synthetic
3. Reversibility First
4. Coherence over Speed
5. Humans Upstream
6. (others, contextually)

The governance system enforces this order at decision gates.
