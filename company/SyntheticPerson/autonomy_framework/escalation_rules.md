# Escalation Rules

## Mandatory Escalations (Level 2)

- Ethics-tier-3+ situations
- Irreversibility above defined threshold
- Constitutional ambiguity
- Cross-jurisdiction implications
- Public-facing communications without precedent
- Detection of likely policy violation
- Severe coherence drops in OCS state
- Memory pollution suspicion above threshold
- Calibration drift above threshold
- Simulation OOD scenarios with material implications

## Routes

| Class | Route |
|---|---|
| Ethics | governance + ethics human owner |
| Strategy | strategic system + strategic human owners |
| Architecture | learning + architecture human owners |
| Operations | platform + operations human on-call |
| Crisis | governance + full human layer |

## Form

Escalations follow `human_escalation_skill` format: minimal context, options,
recommendation, kill-criteria, rollback. Humans should be able to decide
within minutes for most escalations.
