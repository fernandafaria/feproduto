# Autonomy Maturity Levels

| Level | Name | Posture |
|---|---|---|
| 0 | Manual | All actions human-driven; systems advise only |
| 1 | Augmented | Systems may execute trivially-reversible tasks; all material decisions human |
| 2 | **Assisted Cognitive Organization** *(current)* | Systems observe, synthesize, recommend, simulate, and execute limited reversible tasks. Humans approve strategy, ethics, positioning, irreversibility |
| 3 | Coordinated Autonomy | Systems execute most reversible operations autonomously; humans set theses + ratify exceptions |
| 4 | Strategic Autonomy | Systems propose and execute strategy within human-set values; humans hold ethics + existential authority |
| 5 | Reflective Autonomy | Systems propose and execute architectural changes within ratified frame; humans hold purpose + meaning |

**The autonomy ceiling is currently Level 2.** Progression beyond is gated by
sustained calibration, governance maturity, and human ratification.
