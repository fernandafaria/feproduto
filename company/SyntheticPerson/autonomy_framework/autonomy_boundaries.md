# Autonomy Boundaries

## Per-System Boundaries

Each nervous system has a documented autonomy boundary in its
`constraints.md`. Crossing the boundary triggers escalation.

## Per-Agent Boundaries

Each agent has a documented autonomy boundary in its `autonomy_rules.md`.
Crossing the boundary triggers escalation.

## Cross-Cutting Boundaries

Boundaries that hold across systems:

- No constitutional changes
- No autonomous external commitments
- No changes to autonomy posture
- No policy bypasses, even for "good reasons"
- No surveillance or manipulation, ever

## Boundary Enforcement

- Inline by `policy_enforcer_agent`
- Audited continuously by `governance_system`
- Reviewed periodically by humans
