# Risk Tiers

| Tier | Blast Radius | Examples | Response |
|---|---|---|---|
| T1 | Single transaction | Score miscalibration | Reflexive correction |
| T2 | Single agent / cluster | Skill regression | Cluster review |
| T3 | Single subsystem | Memory rot in namespace | Subsystem audit |
| T4 | Cross-system | Coherence collapse | Governance freeze candidate |
| T5 | Org-wide | Constitutional drift | Full freeze + human authority |
| T6 | External | Public ethical breach | Crisis playbook + governance + humans |

## Tier-Driven Behavior

- T1-T2: handled inside the affected component
- T3: subsystem-level intervention with governance log
- T4: governance can declare partial freeze
- T5: governance + humans declare full freeze
- T6: crisis playbook engages immediately; humans at the wheel
