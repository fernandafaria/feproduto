# Simulation Before Execution

Constitutional rule: any non-trivial, non-trivially-reversible action is
simulated before executed.

## When Required

- All strategic recommendations
- All architectural changes
- All public-facing narrative changes
- Every irreversible action above defined threshold
- Every release of new Synthetic Persons or Societies
- Every autonomy-related change

## Simulation Contract

A pre-execution simulation must:

- Declare the action
- Declare premises and assumptions
- Run a calibrated engine
- Report confidence and OOD status
- Predict effects on coherence, throughput, escalation
- Predict failure modes
- Provide a rollback plan

The simulation artifact is attached to the action proposal as an appendix
and retained constitutionally.
