# Collective Memory

The shared organizational memory substrate. Every nervous system, agent, and
skill reads from and writes to here. Identity, coherence, and learning all
rest on this layer.

## Subspaces

| Namespace | Purpose |
|---|---|
| `decisions/` | All material decisions with rationale and authority |
| `simulations/` | Simulation runs, calibration, replays |
| `learnings/` | Learning artifacts, proposals, evolution history |
| `organizational_history/` | The org's longitudinal narrative |
| `research/` | Synthetic studies, methodologies, validity reports |
| `failures/` | Catalogued failures and their lessons |
| `strategic_context/` | Active theses, scenarios, narratives |
| `market_memory/` | Market state, competitor models, demand surfaces |
| `behavioral_models/` | Synthetic Person and Society models |
| `narratives/` | Internal and external narrative artifacts |
| `knowledge_graphs/` | Cross-cutting structured knowledge |
