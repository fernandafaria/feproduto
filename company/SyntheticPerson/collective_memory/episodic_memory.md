# Organizational Episodic Memory

Specific events: a customer interaction, a launch, a near-miss, a market
shift. Episodic memory has high salience for current operations and decays
in salience over time but never in retrievability.

Used by:

- `synthetic_research_system` for grounding studies
- `strategic_intelligence_system` for thesis evidence
- `learning_system` for failure analysis
- `human_layer/` for institutional memory
