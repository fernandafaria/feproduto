# SyntheticPerson.ai

> Simulate before you decide.

SyntheticPerson.ai is an AI-native organization architected from inception as a
distributed cognitive nervous system rather than as a traditional industrial
firm. The company exists to build **Synthetic Persons**, **Synthetic
Societies**, and the **Organizational Cognitive Architectures** required to
simulate, predict, and govern behavior at the scale of individuals, teams,
markets, and civilizations.

## Operating Posture

The organization operates as:

- A continuous collective intelligence
- A distributed nervous system across cognitive subsystems
- An adaptive, self-rewriting structure
- A memory-first organism (organizational memory is the substrate, not a tool)
- A simulation-first decision engine ("simulate before you decide")
- A post-industrial, post-human-centric architecture

## Current Autonomy Posture

**AUTONOMY LEVEL 2 — Assisted Cognitive Organization.**

Cognitive subsystems may observe, synthesize, recommend, simulate, and execute
**limited reversible** tasks. Humans retain authority over strategy, ethics,
positioning, and irreversible change. See `autonomy_framework/` for the full
ladder, escalation rules, and trust mechanics.

## How to Read This Repository

This repository is **not** an org chart. It is the *runtime specification* of a
cognitive organism. Each directory is a subsystem; each file is a contract.
Start at `organization_core/purpose.md`, then read
`organization_core/cognition_model.md`, then explore `nervous_systems/` and
`agents/` in any order — they are designed to be read non-linearly, the same
way they are designed to operate.

## Versioning

The organization itself versions semantically:

- `MAJOR` — change in cognitive architecture or autonomy level
- `MINOR` — addition of a nervous system, cluster, or skill family
- `PATCH` — refinement of an existing module

Current version: **v0.1.0 — Genesis (Autonomy Level 2)**.
