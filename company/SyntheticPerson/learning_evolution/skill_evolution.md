# Skill Evolution

Skills evolve faster than agents. Evolution surfaces:

- Calibration drift triggers recalibration
- Recurring failure modes trigger reasoning-model updates
- Cost-profile drift triggers efficiency rewrites
- Disuse triggers retirement consideration

Skills are easier to retire than agents because their use is composable.
Retirement happens after a deprecation window with migration paths declared.

## Versioning

`{namespace}.{skill}.v{N}`. Backwards-compatible by default.
