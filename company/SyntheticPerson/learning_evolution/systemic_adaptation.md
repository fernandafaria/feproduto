# Systemic Adaptation

Adaptation happens at multiple scopes simultaneously:

- **Sensor adaptation** — sampling rates, source weights
- **Skill adaptation** — calibration, parameter updates
- **Agent adaptation** — context strategy, reasoning model updates
- **Topology adaptation** — subscription rebinds, cluster cohesion
- **Architectural adaptation** — addition/retirement of modules

Each scope has its own approval path (autonomous vs. ratified).

## Stability Anchors

Adaptation is bounded by:

- Constitutional invariants
- Memory continuity
- Reversibility windows
- Coherence audits

Run-away adaptation is detected by OCS and stopped by governance.
