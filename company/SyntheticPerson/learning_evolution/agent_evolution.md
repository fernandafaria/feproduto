# Agent Evolution

Agents evolve through:

- Reasoning-model updates (proposed via learning, ratified via governance)
- Context-strategy refinement (data-driven)
- Skill composition changes (driven by outcomes)
- Promotion (deeper role, expanded sandbox)
- Retirement (function obsoleted or merged)

Evolution proposals carry simulation evidence and rollback plans. No silent
agent changes; every version transition is logged.

## Versioning

`{cluster}.{agent}.v{N}`. Major version on reasoning-model changes; minor
on parameter updates.
