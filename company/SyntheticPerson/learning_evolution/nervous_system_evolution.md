# Nervous System Evolution

Nervous systems evolve slowest. Evolution requires:

- Multi-quarter evidence
- Simulation against the organizational model
- Architectural diff proposal
- Governance ratification
- Human ratification (above tier)
- Phased rollout with rollback windows

## Why So Slow

Nervous systems are deep: their changes ripple. Stability at this layer is
load-bearing for the rest of the org. We accept slowness as a price of
coherence.

## Evolution Triggers

- Sustained mismatch between system function and environment variety
- Persistent failure to absorb new responsibilities
- Architectural diff proposals from learning_system with evidence
- Constitutional changes that require system-level alignment
