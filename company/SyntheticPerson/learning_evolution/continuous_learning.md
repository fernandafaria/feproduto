# Continuous Learning

Learning is continuous, not periodic. It happens at five levels simultaneously:

| Level | Loop |
|---|---|
| Skill | Per-invocation calibration |
| Agent | Per-transaction outcome scoring |
| Cluster | Cluster-cohesion + outcome metrics |
| Subsystem | Per-recommendation acceptance + outcome |
| Organization | OII + longitudinal reflection |

## Inputs

- Outcome data
- Failure memory
- Calibration histories
- Coherence audits
- Human feedback (sparse but high-salience)

## Outputs

- Updated calibrations
- Promoted patterns
- Retired primitives
- Architectural diff proposals
- Constitutional review prompts
